1設定 LINE 官方帳號

1-1 開通 Messaging API

  1. 前往 LINE Developers Console，用你的 LINE 帳號登入
  2. 點選 Create a Provider（輸入你的品牌名稱）
  3. 點選 Create a Messaging API channel
  4. 填寫：
    • Channel name：你的品牌名稱
    • Channel description：簡短說明
    • Category / Subcategory：選擇最接近的分類
  5. 點選 Create

1-2 取得金鑰

建立完成後：

  1. 進入剛建立的 Channel → Basic settings 頁籤
    找到 Channel Secret → 點 Copy 複製並記下來
  2. 切換到 Messaging API 頁籤
    滑到最下方 Channel access token → 點 Issue
    複製這串 Token 並記下來
⚠️ 這兩組資訊非常重要，請妥善保管，不要分享給不相關的人，也不要截圖上傳到公開平台。

1-3 確認設定

在 Messaging API 頁籤確認以下設定：

  • Use webhook：請開啟
  • Auto-reply messages：請關閉（否則會和 Bot 衝突）
  • Greeting messages：可依喜好開啟或關閉

2設定 Telegram 接收通知

Telegram 是你接收「需要人工回覆」通知的工具，免費且即時。

2-1 建立 Telegram Bot

  1. 在 Telegram 搜尋 @BotFather，點選開始對話
  2. 傳送 /newbot
  3. 依指示輸入 Bot 名稱和 Username（Username 必須以 bot 結尾，例如 YourShop_bot）
  4. 建立成功後會收到一串 Token，格式如：123456789:ABCDefGHI...
  5. 複製這串 Token 並記下來

2-2 取得你的 Chat ID

  1. 在 Telegram 搜尋 @userinfobot，點選開始並傳送任意訊息
  2. 它會回覆你的 ID 數字（例如 123456789）
  3. 記下這個數字

2-3 啟動 Bot

找到你剛建立的 Bot（搜尋你取的 Username），點選 Start 開始對話。

ℹ️ 必須先點 Start 開始對話，Bot 才能傳訊息給你。這個步驟很容易忘記。

3建立 Google Sheets 記錄表

3-1 建立試算表

  1. 開啟 Google Sheets，建立一份新試算表
  2. 將試算表命名為你想要的名稱（例如「LINE對話記錄」）
  3. 複製試算表的 URL，取出其中的 Sheet ID：
https://docs.google.com/spreadsheets/d/【這一段就是 Sheet ID】/edit

3-2 共享給系統帳號

我們會提供你一個 Service Account Email（格式如 xxx@project.iam.gserviceaccount.com）

  1. 在試算表右上角點 共用
  2. 輸入該 Email，權限設為編輯者
  3. 點 傳送

4將資訊交給我們

請將以下資訊提供給我們，我們幫你完成部署：

LINE_CHANNEL_SECRET       = （Step 1-2 取得）
LINE_CHANNEL_ACCESS_TOKEN = （Step 1-2 取得）
LINE_OA_BASIC_ID          = 你的 LINE OA Basic ID（格式：@xxx）

TELEGRAM_BOT_TOKEN        = （Step 2-1 取得）
TELEGRAM_CHAT_ID          = （Step 2-2 取得）

GOOGLE_SHEET_ID           = （Step 3-1 取得）
GOOGLE_SHEET_TAB          = LINE對話記錄（或你自訂的工作表名稱）
🔒 請使用私訊或加密方式傳送，避免在公開群組或 Email 主旨等公開管道分享這些金鑰。

5完成設定後的確認

我們完成部署後，請做以下驗收確認：

  • 在你的 LINE 官方帳號傳送一則測試訊息
  • 確認 Bot 有自動回覆
  • 傳送「我要預約」，確認 Telegram 有收到通知
  • 打開 Google Sheets，確認對話有被記錄下來
✅ 四項都通過就代表部署成功，Bot 已可正式上線！

日常使用說明

何時會收到 Telegram 通知？

情況說明
顧客填寫預訂資訊顧客提交了預約表單內容
對話超過設定輪數超過 10 輪對話仍未解決
顧客表示已付款Bot 偵測到付款確認語句
緊急優先訊息Bot 判斷需要立即人工介入
逾時未回覆警報顧客等待超過 4 小時未獲回覆

如何接手人工回覆？

收到 Telegram 通知後，直接進入你的 LINE 官方帳號管理介面，找到該顧客手動回覆即可。Bot 不會和你的手動回覆衝突。

如何調整 Bot 的回覆語氣或知識庫？

請聯絡我們調整，通常 1–2 個工作天內完成更新。
