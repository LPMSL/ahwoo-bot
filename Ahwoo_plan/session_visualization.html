中途遇到的卡點

Blockers
Raw 檔案有極慢 case 少數 CSV 讀取時會極慢，曾出現單檔 300 秒級延遲，已改成 bytes decode 方式並加入慢檔跳過策略。
本地 Python 是 3.9 claude_handler.py 原本使用 3.10 的型別語法，已補上 future annotations 相容處理。
本地沒有網路與 Anthropic key 因此 current_bot replay 目前只能反映 fallback 行為，而不是 live Claude 生成結果。
Telegram 模組不在本地 Python 環境 API 失敗通知原本會讓 replay 中斷，現在已改成通知載入失敗時略過，不影響評測流程。

Next Move
